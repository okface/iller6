# Road Sign Images (Vägmärken)

This directory should contain Swedish road sign images referenced in körkortsteori questions.

## Required Images

The following images are currently referenced in questions:
- `vagmarke_barn.png` - Warning sign for children
- `vagmarke_stopplikt.png` - Stop sign

## Image Guidelines

1. **Format**: PNG or JPG
2. **Size**: Recommended 300x300 to 600x600 pixels
3. **Naming**: Use pattern `vagmarke_[description].png`
4. **Source**: 
   - Official: Transportstyrelsen (https://www.transportstyrelsen.se/sv/vagtrafik/Vagmarken/)
   - Wikipedia Commons (ensure proper licensing)
   - Create simple SVG representations

## Adding Images

When you generate new questions with road signs:
1. Note which image filenames are referenced
2. Download or create those images
3. Place them in this directory
4. Ensure filenames match exactly what's in the YAML

## Copyright Note

When using images, ensure you have the right to use them. Swedish road signs are regulated by Swedish Transport Agency (Transportstyrelsen). For educational purposes, you may:
- Use official images from Transportstyrelsen with proper attribution
- Use Creative Commons images from Wikipedia
- Create your own simplified versions for educational use
